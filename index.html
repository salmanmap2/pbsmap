API

Developer API

PBS Map একটি সম্পূর্ণ REST API প্রদান করে। আপনার নিজস্ব অ্যাপ্লিকেশন তৈরি করুন আমাদের API ব্যবহার করে।

  • 🔑
    API Key Authentication

    প্রতিটি ব্যবহারকারীর নিজস্ব API key আছে

  • 📡
    RESTful Endpoints

    মিটার, রিডিং, নোট, অফিস সব কিছুর API

  • 🔄
    Incremental Sync

    last_time parameter দিয়ে শুধু নতুন ডেটা নিন

API ব্যবহার শুরু করুন →
api-example.js 
// মিটার লিস্ট নিন
const res = await fetch(
  '/api/meter/all', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    office_id: '12101',
    last_time: '2024-01-01T00:00:00'
  })
});

const { data } = await res.json();
// data = [{ account_id, meter_number, ... }]
